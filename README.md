# Web3 Liquidation Risk Monitor

A professional Web3 reference project that demonstrates how decentralized lending risk systems can be modeled using clean and modular Web3 architecture.

This repository focuses on simulating loan positions, monitoring collateral ratios, detecting liquidation thresholds, and generating risk analytics in a beginner-friendly and production-style format.

---

## Supported Networks

- Ethereum  
- Base  
- Polygon  
- Arbitrum  

---

## Project Goals

- Simulate DeFi loan positions  
- Monitor collateral health  
- Detect liquidation thresholds  
- Analyze risk exposure  
- Teach lending risk architecture  

---

## Architecture Overview

This project uses a root-only structure where each file handles a single responsibility such as network access, loan registry, risk engine, threshold analyzer, and analytics reporting.

---

## Features

- Multi-chain RPC support  
- Loan position simulation  
- Collateral ratio calculation  
- Liquidation risk detection  
- Clean console-based reporting
